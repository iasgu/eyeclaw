视频与分析参数

视频分析现在会严格参照监听器会话，不再使用纯均匀抽帧逻辑。

说明：监听器扩展负责采集点击、输入、截图和录屏，供“分析监听会话/分析视频流程”使用；“执行实时流程”默认使用 Browser Use 智能执行器，旧版本地回放只作为备用。`连接已打开的 Edge` 主要用于查看当前本机 CDP 浏览器状态，分析出步骤后可直接执行。

快速使用说明

1. 启动监听器 打开 `edge://extensions`，加载 `browser_listener_extension` 目录。
2. 真实操作页面 在 Edge 里按真实流程点击，监听器会记录关键事件与截图。
3. 分析监听会话 点击"分析监听会话"，系统会优先用监听器引导关键帧理解。
4. 结合视频验证 如果要分析 `website.mp4`，必须先有监听会话，再点"分析视频流程"。
5. 监听模式如何执行任务 监听模式本身不直接执行网页操作，它的作用是先采集过程并分析出步骤。要执行任务，需要把分析结果保存成技能，再在用户端任务页或控制台执行，执行时仍然走 CDP。
